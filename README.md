## 当前促销方式：三条路径

**路径一：年付直接省**
几乎所有套餐切到年付都有折扣，折算下来相当于免 1–2 个月。不需要折扣码，结账时选年付计费周期即可。

**路径二：折扣码叠加**
DMIT 不定期在官方 Twitter（@DMIT_Inc）和工单系统发放折扣码。常见的循环折扣码格式是固定百分比，适用于特定机房或线路。折扣码在结账页面的 Promo Code 栏填入，系统会实时显示折后价。

**路径三：限时特价套餐**
部分节日（黑五、新年）会推出限量特价套餐，库存卖完即止。这类套餐通常不在常规套餐列表里，需要关注官方公告。

## 三个常见误解，直接说清楚

**误解一："DMIT 贵，不值"**
贵是真的。LAX EB 系列月付 $3.9 起，但 Pro 系列月付 $6.9 起，比某些低价 VPS 贵一倍。差价买的是线路质量——CN2 GIA 和软银线路在晚高峰的表现和普通 BGP 线路不在一个量级。如果你的使用场景对延迟不敏感，确实没必要选 DMIT。

**误解二："流量超了就断线"**
不是断线，是限速到 1Mbps。服务器还在跑，SSH 还能连，只是带宽被压下来了。下个计费周期自动恢复，或者手动购买额外流量包。

**误解三："香港节点最好"**
不一定。香港 Pro 延迟最低，但价格也最高，且库存经常告罄。如果你在华南，香港确实有优势；华北用户选洛杉矶 CN2 GIA 有时候反而更稳，因为绕开了国内骨干网的拥堵点。

## 适合谁 / 不适合谁

**适合：**
- 需要稳定低延迟回国线路的开发者、远程办公用户
- 对晚高峰丢包率有要求的用户（游戏加速、视频会议）
- 愿意为线路质量付溢价、不想频繁换机器的用户
- 需要香港、日本、美国多地部署的团队

**不适合：**
- 预算有限、只需要跑基础服务的个人用户（EB 系列可以考虑，Pro 系列没必要）
- 对线路质量无感、只看配置参数的用户
- 需要大量存储空间的用户（DMIT 的 SD 容量普遍偏小）

## FAQ

**Q：DMIT 支持支付宝付款吗？**
支持。结账页面支持支付宝、PayPal、信用卡，国内用户用支付宝最方便，不需要绑定境外卡。

**Q：DMIT 有试用期吗？**
没有免费试用，但部分套餐支持 72 小时内退款。购买前建议先确认目标套餐是否在退款政策覆盖范围内，可以在下单前开工单询问。

**Q：折扣码在哪里找？**
官方渠道是 DMIT 的 Twitter 账号（@DMIT_Inc）和官网公告页。第三方聚合站的折扣码很多已经过期，直接去官方渠道确认最准。

**Q：香港套餐为什么经常显示 Out of Stock？**
香港 CN2 GIA 机房资源有限，热门套餐库存紧张是常态。可以在官网开启库存通知邮件，有货时第一时间收到提醒。

**Q：DMIT 和搬瓦工比哪个好？**
两家都做 CN2 GIA，定位接近。DMIT 香港节点是搬瓦工没有的优势；搬瓦工在洛杉矶有更多套餐选择，价格梯度更细。如果你主要用洛杉矶节点，两家都值得对比；如果需要香港节点，DMIT 是更直接的选择。

**Q：超流量之后怎么处理？**
速度限制到 1Mbps，下个计费周期自动恢复满速。也可以在后台购买额外流量包，按需添加，不需要升级套餐。
